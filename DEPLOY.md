## 🌐 Deploy to Vercel (2 Minutes!)

### Step 1: Go to Vercel
Visit: https://vercel.com

### Step 2: Sign Up
Click "Sign Up" → "Continue with GitHub"

### Step 3: Import Project
1. Click "Add New..." → "Project"
2. Find `shreyacricketacademy` repository
3. Click "Import"

### Step 4: Deploy
1. Project Name: `shreya-cricket-academy` (or any name)
2. Framework: Other
3. Click "Deploy"
4. Wait 30-60 seconds
5. Done! 🎉

### Your Live URL:
```
https://shreya-cricket-academy.vercel.app
```
